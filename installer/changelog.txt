Fz10m changelog
https://github.com/jackharrhy/Fz10m

2026-04-25 - v0.6.0
  - Additive harmonic mode: 16 harmonic sliders with Draw/Additive toggle
  - Harmonic presets: selecting a waveform preset in additive mode sets the harmonic recipe
  - Mode selector for wavetable area (Draw for waveform drawing, Additive for harmonics)

2026-04-25 - v0.5.0
  - Filter mode selector: LowPass, HighPass, BandPass, Notch, Peak
  - Lo-fi pre/post filter toggle

2026-04-25 - v0.4.0
  - Per-voice filter envelope (ADSR modulating cutoff with bipolar amount control)
  - Wavetable preset waveforms: sine, triangle, saw, square, pulse, double sine, saw/pulse, random
  - Two-row knob layout (Synth + ADSR / Filter Env + LoFi)

2026-04-25 - v0.3.0
  - 8-voice polyphony (matching the real Casio FZ-10M)
  - Reset button to restore all knobs and wavetable to defaults
  - Saner minimum envelope times to prevent clicks (attack 5ms, decay/release 20ms)
  - Per-version release notes on GitHub releases

2026-04-25 - v0.2.1
  - Fix knobs not showing correct positions on launch

2026-04-25 - v0.2.0
  - Wavetable now persists across GUI close/reopen and project save/load
  - Versioned state serialization for forward-compatible project files

2026-04-25 - v0.1.0
  - Per-voice lo-fi stage: sample-rate hold, bit-depth quantization, character blend
  - Stepped filter coefficient updates for FZ-inspired zipper character
  - UI knobs grouped into Synth / ADSR / LoFi sections
  - Replaced placeholder metadata with real Grovepark/Fz10m values
  - Release now prompts for bump type and confirmation before pushing
  - `just app` recipe for quick standalone testing

2026-04-24 - v0.0.1 initial release
  - Drawable 128-point wavetable oscillator
  - ADSR amplitude envelope
  - SVF low-pass filter with cutoff and resonance
  - Builds as VST3, AU, CLAP on macOS and VST3, CLAP on Windows
